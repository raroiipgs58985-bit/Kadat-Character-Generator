# Схема записи

Каждая самостоятельная запись каталога содержит:

- `id` — стабильный технический идентификатор;
- `name` — название из исходной таблицы;
- `category` — категория записи;
- `description` — исходное описание, если оно присутствует;
- `cost`, `woundBonus` или `bonusPoints` — число либо `null`, если значение в источнике отсутствует;
- `characteristicsText`, `skillsText`, `talentsText`, `equipmentText`, `effectText`, `restrictionsText` — исходные формулировки соответствующих колонок;
- `source.sheet`, `source.row`, `source.range` — ссылка на положение записи в исходной таблице.

Структурированные поля создаются только там, где значение однозначно указано в таблице. Текст описания сам по себе не используется для добавления отсутствующих бонусов, цены или эффектов.
