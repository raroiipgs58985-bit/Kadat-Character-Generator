# Генератор персонажей для Кадата

Браузерный генератор персонажей для пользовательской системы к100.

## Текущая версия

`v0.9-advancement` — тестовая сборка с полной основой персонажа и покупкой развития за очки опыта.

Уже реализовано:

- характеристики НС, НР, СЛ, ВН, ЛВ, ИН, СВ, ВС и ОЩ;
- распределение 100 очков или броски 2к10;
- два переноса по 5 очков между любыми характеристиками;
- восемь рас, двенадцать родных миров и 37 заполненных специальностей;
- фильтрация специальностей по расе и полу;
- обязательные выборы вариантов, разделённых знаком `/`;
- повторный навык даёт +5, максимум +30;
- повторный талант происхождения заменяется на +25 ОО;
- каталог из 30 навыков со специализациями;
- каталог из 242 талантов с уровнями, требованиями и описаниями;
- пулы владения оружием, ремёсел, знаний, языков, управления, сопротивляемости и других специализаций;
- покупка характеристик по +5 за 250 / 500 / 750 / 1000 ОО;
- покупка и улучшение навыков: 200 (+0), 350 (+10), 500 (+20), 750 (+30) ОО;
- покупка талантов за 250 / 500 / 750 ОО по уровню;
- автоматическое вычитание стоимости специальности и покупок из доступного опыта;
- пересчёт ран после повышения Выносливости и Силы воли;
- поиск и фильтрация талантов;
- отмена отдельных покупок и полный сброс развития;
- итоговый раздел с приобретённым развитием;
- адаптивный интерфейс для телефона.

Формула ран:

```text
Бонус ран расы + бонус родного мира + бонус специальности + (БСВ + БВН) × 2
```

Сложные текстовые требования талантов выводятся полностью. Генератор автоматически проверяет требования по характеристикам и некоторые расовые ограничения; остальные требования пользователь сверяет по тексту карточки таланта.

Пока не добавлены незаполненные в исходной таблице специальности: Банши, Огненный дракон, Провидец, Сновидец, Певец кости, Ведьма и Ученик-гомункла.

## Проверка

В репозитории настроен GitHub Actions, который проверяет синтаксис JavaScript, наличие подключённых файлов и целостность каталогов навыков и талантов.
