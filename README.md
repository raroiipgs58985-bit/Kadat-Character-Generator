# Генератор персонажей для Кадата

Браузерный генератор персонажей для пользовательской системы к100.

## Текущая версия

`v0.1-core` — техническая сборка для проверки базовой математики.

Уже реализовано:

- характеристики НС, НР, СЛ, ВН, ЛВ, ИН, СВ, ВС и ОЩ;
- плановое распределение 100 очков с лимитом 20 в характеристику;
- случайная генерация 2к10;
- применение модификаторов расы, родного мира и специальности;
- расчёт бонуса характеристики через десятки итогового значения;
- расчёт ран по формуле:

```text
Бонус ран расы + бонус родного мира + бонус специальности + (БСВ + БВН) × 2
```

- повторное получение навыка даёт +5, максимум +30;
- повторный талант заменяется на +25 опыта;
- адаптивный интерфейс для телефона.

Пока в базе находятся только тестовые родной мир и специальность. После проверки ядра они будут заменены полной таблицей правил.

## Запуск

Открой `index.html` в браузере или опубликуй ветку через GitHub Pages.

## Ветки

- `main` — проверенная версия;
- `feature/v0.1-core` — текущая разработка первой версии.
