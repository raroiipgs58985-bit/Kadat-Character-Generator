# Генератор персонажей для Кадата

Браузерный генератор персонажей для пользовательской системы к100.

## Текущая версия

`v0.5-races` — первая тестовая сборка с полной базой рас из исходной таблицы правил.

Уже реализовано:

- характеристики НС, НР, СЛ, ВН, ЛВ, ИН, СВ, ВС и ОЩ;
- плановое распределение 100 очков с лимитом 20 в характеристику до переносов;
- случайная генерация 2к10;
- до двух переносов по 5 очков из любой характеристики в любую другую;
- применение модификаторов расы, родного мира и специальности;
- расчёт бонусов характеристик и ран;
- повторное получение навыка даёт +5, максимум +30;
- повторный талант заменяется на +25 опыта;
- адаптивный интерфейс для телефона;
- восемь рас: Человек, Эльдар, Друкхари, Астартес, Огрин, Ратлинг, Скват и Зверолюд;
- расовые характеристики, раны, навыки, таланты, особенности, особые правила и стартовое снаряжение;
- отображение расовой информации в итоговой карточке персонажа.

Формула ран:

```text
Бонус ран расы + бонус родного мира + бонус специальности + (БСВ + БВН) × 2
```

Родные миры и специальности пока остаются тестовыми. Следующий этап — перенос полной базы родных миров.
