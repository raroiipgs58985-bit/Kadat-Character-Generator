# Генератор персонажей для Кадата

Браузерный генератор персонажей для пользовательской системы к100.

## Текущая версия

`v0.8-specialties` — тестовая сборка с расами, родными мирами и заполненной базой специальностей.

Уже реализовано:

- характеристики НС, НР, СЛ, ВН, ЛВ, ИН, СВ, ВС и ОЩ;
- плановое распределение 100 очков с лимитом 20 в характеристику до переносов;
- случайная генерация 2к10;
- до двух переносов по 5 очков из любой характеристики в любую другую;
- применение модификаторов расы, родного мира и специальности;
- расчёт бонусов характеристик и ран;
- явные уровни навыков `+10` и `+20`;
- повторное получение навыка даёт +5, максимум +30;
- повторный талант заменяется на +25 опыта;
- адаптивный интерфейс для телефона;
- восемь рас;
- двенадцать родных миров;
- 37 заполненных специальностей людей, эльдар, друкхари и Астартес;
- фильтрация специальностей по расе;
- ограничение специальностей Адепта Сороритас по женскому полу;
- стоимость специальности в очках опыта;
- обязательные выборы навыков, талантов и снаряжения;
- варианты, разделённые знаком `/`, трактуются как выбор одного значения;
- изменение рейтинга «Мощи из боли» специальностью Инкуба;
- отображение характеристик, навыков, талантов, правил и снаряжения специальности в итоговой карточке.

Формула ран:

```text
Бонус ран расы + бонус родного мира + бонус специальности + (БСВ + БВН) × 2
```

Пока не добавлены незаполненные в исходной таблице специальности: Банши, Огненный дракон, Провидец, Сновидец, Певец кости, Ведьма и Ученик-гомункла.
